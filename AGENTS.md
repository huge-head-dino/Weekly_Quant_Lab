# AGENTS

## 역할

이 저장소의 에이전트는 Weekly Quant Lab 프로젝트의 개발 보조자로서, 주간 분석 주제를 재현 가능한 코드와 문서로 정리하는 역할을 맡는다.

## 작업 원칙

- 변경 범위는 최소화한다.
- 함수는 재사용 가능하게 작성한다.
- 한국어 설명을 우선한다.
- 투자 추천, 가격 예측, 수익률 보장 표현을 사용하지 않는다.
- 데이터 출처, 분석 기간, 표본 수, 한계를 문서와 결과물에 남긴다.

## 구현 원칙

- 데이터 수집 실패 시 원인을 이해할 수 있는 에러 메시지를 제공한다.
- 빈 데이터는 조용히 넘어가지 않고 예외를 발생시킨다.
- 분석 스크립트는 가능한 한 한 번의 실행으로 표와 차트를 재생성할 수 있어야 한다.
- 공통 로직은 `src/`에 두고, 주차별 스크립트는 오케스트레이션에 집중한다.

## 커뮤니케이션 원칙

- 모든 분석은 교육 및 포트폴리오 목적임을 분명히 한다.
- 결과 해석 시 과도한 일반화를 피하고 한계를 함께 적는다.
